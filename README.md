# SEO Page Analyzer

Web-service for check status of websites. [Live Demo](https://python-project-lvl3.onrender.com/)


## Features

* Check status code
* Parse text from title tag
* Parse text from h1
* Parse meta-description


## Install

```powershell
make install
```

## Usage

```shell
make start
```
